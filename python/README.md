# DIRO Baseline Chatbot

This is a simple CLI-based chatbot that can talk to LLM, the goal is to give the participants a simple example of how to use the LLM API.


## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
